Animate

HTML modeling is a way to describe real-world concepts visually. HTML modeling is different from XML modeling because the focus is on modeling for presentation purposes. HTML modeling is important for all page elements that are presented to the visitor.

Many problems can be avoided or fixed by fixing the HTML model and by considering how to improve the HTML model. Better HTML models will improve the ability to use CSS to better stylize a page, and better HTML models will help improve the ability to use JavaScript to interact with users.

A marker has these elements: cap, handle, label, ink well, and felt, but for presentation purposes we might also need to add a cap shadow element and a label element. Conclusions

HTML modeling is an important step in quality Web application development. All page content can benefit from the power of HTML by carefully considering how to arrange the HTML in a way that best models the page elements for presentation purposes.
