Modeling a Felt Marker Using HTML

This page demonstrates HTML modeling.

HTML modeling (similar to XML modeling) is a way to describe real-world concepts visually. In this case, the focus is on the presentation. Better HTML models will help the CSS to better stylize a page, and better HTML models will help the JavaScript to interact with users.

You start with the HTML to describe things in a structured manner. A marker has these elements: cap, handle, label, ink well, and felt, where the felt goes inside the well, and the well fits inside the handle.

CSS and JavaScript will then interact with the model by stylizing or altering the HTML.

Then, CSS can be used to help visually enhance the information. Improving the HTML model will help the CSS to effectively enhance the information.

Conclusions:

All page content can benefit from the power of HTML by carefully considering the layering of HTML elements. HTML modeling is important for all page elements that are presented to the visitor.

HTML modeling is an important step in quality Web development and Web app development. Many problems can be avoided or fixed by starting from the HTML model, and considering how to improve the HTML model.
